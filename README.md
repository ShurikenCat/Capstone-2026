An AI-assisted academic productivity app that parses syllabi, creates and manages schedules, tracks assignments, and provides chatbot-based student support.

HOW TO RUN THE PROJECT

1. Download Android Studio: https://developer.android.com/studio

2. Open a terminal in Android Studio and run the following command:
`git clone https://github.com/ShurikenCat/Capstone-2026`

3. Create an emulator in Android Studio
   - Click Device Manager in the top right
   - Add a new device
   - Create virtual device
   - Select which device to use, this project was developed using the medium phone

4. Make sure the required libraries are installed

5. Build and run the project
